第11讲：无线信道建模——多径、多普勒与信道模型

一、无线信道：工程师的噩梦与舞台

问任何一个做基带算法的工程师：“你最头疼什么？” 十有八九会说：信道。

发射机的每一个模块，从 IFFT 到加窗，都是可预期的确定性操作，调通了就是调通了。但信道不一样——它是随机的、时变的、环境相关的。同一栋楼里，从走廊换到会议室，信道特性可能判若云泥。

理解无线信道，不是为了“对付”它，而是为了针对信道特性，设计出真正有效的算法。OFDM 本身的诞生，就是对频率选择性信道的一次深刻应答。

二、多径信道的数学模型

2.1 冲激响应描述

现实中的无线传播路径不止一条。信号从发射天线出发，经过建筑物、地面、树木的反射、散射、绕射，以多条路径、不同延迟、不同相位到达接收天线。

这种多径效应用时变冲激响应（Time-Varying Impulse Response）来描述：

\[ h(\tau,t)=\sum_{l=0}^{L-1}\alpha_l(t)\cdot\delta(\tau-\tau_l) \]

其中：

  • $L$：多径数量（Path数）
  • $\alpha_l(t)$：第 $l$ 条路径的复数增益（包含幅度衰减和相位旋转），随时间 $t$ 缓慢变化
  • $\tau_l$：第 $l$ 条路径的传播时延
  • $\delta(\cdot)$：狄拉克冲激函数

接收信号 $r(t)$ 是发送信号 $s(t)$ 与信道冲激响应的卷积：

\[ r(t)=\int h(\tau,t)\cdot s(t-\tau)\,d\tau+n(t) \]

其中 $n(t)$ 是加性高斯白噪声（AWGN）。

2.2 四个关键参数

无线信道有四个必须掌握的“描述子”：

参数 定义 典型量级 工程含义
时延扩展 $T_d$ 多径时延的均方根（RMS DS） 室内 10-50 ns，室外 0.1-5 μs 决定 CP 长度下限
相干带宽 $B_c$ $\approx 1/(5T_d)$ 室内 >1 MHz，室外 50-200 kHz 子载波带宽 $\Delta f \ll B_c$
多普勒扩展 $f_D$ 移动速度引起的频率展宽 步行 5 Hz，高铁 1000+ Hz 决定信道的变速率
相干时间 $T_c$ $\approx 0.423/f_D$ 静止 >10 s，高铁 ms 信道估计更新周期设计依据

物理直觉：

  • 时延扩展大 → 频率选择性强 → OFDM 子载波要更窄（$\Delta f$ 更小，或 CP 更长）
  • 多普勒扩展大 → 信道变化快 → 需要更密集的导频、更快的同步更新

2.3 平坦衰落 vs 频率选择性衰落

这是一个高频考点，也是工程设计中的判决条件：

信号带宽 W  vs  相干带宽 Bc

W << Bc  →  平坦衰落（Flat Fading）：整个信号带宽内 H(f) ≈ 常数
W >> Bc  →  频率选择性衰落（Frequency-Selective Fading）：H(f) 随 f 剧烈变化

OFDM 的精髓正在于此：虽然系统总带宽 $W$ 远大于 $B_c$，但每个子载波的带宽 $\Delta f \ll B_c$，从而每个子载波经历的是平坦衰落——均衡问题退化为一个复数乘法！

三、统计衰落模型

3.1 瑞利衰落与莱斯衰落

当多径分量数量很多、且无主导路径（直射径LOS）时，每条路径的实部和虚部各自服从高斯分布，信道包络服从瑞利分布（Rayleigh Distribution）：

\[ p(r)=\frac{r}{\sigma^2}e^{-r^2/(2\sigma^2)},\qquad r\ge 0 \]

当存在强直射径（LOS component）时，信道包络服从莱斯分布（Rician Distribution），由莱斯因子 $K$ 刻画 LOS 功率与散射功率之比：

\[ K=\frac{A^2}{2\sigma^2} \]

$K\to0$ 时退化为瑞利衰落，$K\to\infty$ 时退化为 AWGN 信道。

实际场景对照：

  • 室外宏蜂窝：无LOS，多用瑞利模型
  • 室内、毫米波：常有LOS，用莱斯模型（$K=5\sim15\,\mathrm{dB}$）
  • 卫星链路：强LOS，$K\gt20\,\mathrm{dB}$

3.2 MATLAB仿真：瑞利衰落信道冲激响应

下面用 MATLAB 生成一个典型的多径瑞利衰落信道，参数参考 3GPP ETU（Extended Typical Urban）模型：

%% 多径瑞利衰落信道仿真（参考3GPP ETU）
clear; clc;

% ETU信道参数（部分路径）
delays_us = [0, 0.05, 0.12, 0.2, 0.23, 0.5] * 1e-6;  % 时延 [s]
power_dB = [-1.0, -1.0, -1.0, 0.0, 0.0, 0.0];        % 功率 [dB]
power_lin = 10.^(power_dB/10);
power_lin = power_lin / sum(power_lin);               % 归一化

Fs = 30.72e6;   % 采样率（LTE 20MHz）
numPaths = length(delays_us);
numSamples = 1024;

% 生成各路径的复高斯增益（瑞利模型）
h_paths = zeros(numPaths, numSamples);
for k = 1:numPaths
    h_paths(k,:) = sqrt(power_lin(k)/2) * (randn(1,numSamples) + 1j*randn(1,numSamples));
end

% 构建信道冲激响应（离散延迟）
delay_samples = round(delays_us * Fs);
maxDelay = max(delay_samples) + 1;
h_cir = zeros(maxDelay + 1, numSamples);
for k = 1:numPaths
    h_cir(delay_samples(k)+1, :) = h_paths(k,:);
end

% 绘制单个时刻的功率延迟谱（PDP）
figure;
stem(delay_samples/Fs*1e6, 10*log10(power_lin), 'filled');
xlabel('时延 (\mus)'); ylabel('功率 (dB)');
title('ETU 功率延迟谱 (PDP)');
grid on;

运行上述代码，可以看到 ETU 信道在 $0\sim0.5\,\mu\mathrm{s}$ 范围内的 6 条多径，覆盖时延远短 LTE 中 CP 长度（$4.7\,\mu\mathrm{s}$，约144个采样点），因此 LTE 中 CP 长度的设计对 ETU 信道有充裕的保护余量。

四、标准信道模型

仿真结果只有在业界认可的标准信道模型下才具有可比性。主要有两个体系：

4.1 3GPP 传统模型（4G时代）

模型名称 全称 时延扩展 典型场景
AWGN 加性高斯白噪声 0 无衰落基准
EPA Extended Pedestrian A 45 ns 低速步行，室内/郊区
EVA Extended Vehicular A 357 ns 中速车载
ETU Extended Typical Urban 991 ns 高速城市宏蜂窝

这些模型规定了多径时延和相对功率，每条路径的复增益为独立瑞利随机过程，多普勒频率设置对应不同移动速度（EPA：5 km/h，EVA：70 km/h，ETU：300 km/h）。

4.2 5G NR标准模型：CDL/TDL

5G NR引入了两类新模型（TR 38.901）：

TDL（Tapped Delay Line）：非空间信道，适合链路级仿真，有 TDL-A/B/C/D/E 五类

CDL（Clustered Delay Line）：带角度扩展的空间信道，适合MIMO系统评估，有 CDL-A/B/C/D/E 五类

以CDL-A为例（NLOS城市宏蜂窝）：

CDL-A 关键参数:
- 多径数：23 clusters × 20 rays
- RMS DS：归一化后约 5~300 ns（可缩放）
- RMS ASA（方位角扩展）：65°
- 路径损耗模型：UMa-NLOS

CDL模型支持时延缩放，可以通过一个缩放因子 $D_s$ 调整时延扩展以适应不同频段（如FR1 的 6 GHz以下 vs FR2的毫米波）：

\[ \tau_l^{\mathrm{scaled}}=D_s\cdot\tau_l^{\mathrm{nominal}} \]

4.3 选哪个模型?

研究目的                    推荐模型
────────────────────────────────────
算法基础性能验证（无衰落）  AWGN
4G LTE链路仿真             EPA / EVA / ETU
5G NR链路级仿真（单天线）   TDL-A/B/C
5G NR系统级/MIMO仿真       CDL-A/B/C/D/E
毫米波/mmWave专项研究      CDL with FR2缩放

五、多普勒效应：信道的“时间脸”

前面说的多径主要影响频域（频率选择性），而多普勒效应影响的是时域（时间选择性）。

当移动台以速度 $v$ 运动时，从方位角 $\theta$ 方向来的散射分量产生的多普勒频移为：

\[ f_d=\frac{v}{\lambda}\cos\theta=\frac{vf_c}{c}\cos\theta \]

其中 $\lambda=c/f_c$ 为波长，$c$ 为光速，$f_c$ 为载波频率。

几个典型数值（$f_c=3.5\,\mathrm{GHz}$，5G NR Sub-6G频段）：

移动速度 最大多普勒频移 $f_{D,\text{max}}$ 相干时间 $T_c$
步行 3 km/h 9.7 Hz 43.6 ms
车速 120 km/h 389 Hz 1.09 ms
高铁 360 km/h 1167 Hz 0.36 ms

在 OFDM 系统中，子载波间隔 $\Delta f$ 必须远大于多普勒频移，否则子载波间正交性被破坏（产生 ICI）。5G NR FR1 的最大子载波间隔是 60 kHz，对应相干时间约 0.36 ms，正好能覆盖高铁场景（需配合 DMRS 密度加密）。

六、从信道模型到系统设计参数

建立了信道模型的直觉之后，我们可以把信道参数直接映射到 OFDM 系统的关键设计决策：

信道特性                    OFDM设计对应
──────────────────────────────────────────
时延扩展 T_d              → CP长度 > T_d（LTE：4.7μs，5G NR：多种SCS配置）
相干带宽 B_c              → 子载波间隔 Δf << B_c（每子载波看到平坦衰落）
多普勒扩展 f_D            → OFDM符号持续时间 T_s << 1/f_D（信道在一符号内近似静止）
相干时间 T_c              → 导频密度：每 T_c 内至少一个导频符号
信道阶数 L（多径数）       → 信道估计算法复杂度 O(L^2) 或 O(L log L)

这张映射表非常实用——它解释了为什么5G NR引入了 Flexible Numerology（15/30/60/120/240 kHz五种子载波间隔）：不同场景下信道的时延/多普勒特性差异极大，一套固定参数无法兼顾所有场景，必须灵活配置。

七、小结

本讲从物理机制出发，系统梳理了无线信道建模的核心框架：

  1. 多径信道模型：冲激响应 $h(\tau,t)$，包含时延、幅度、相位三个维度
  2. 四大描述参数：时延扩展、相干带宽、多普勒扩展、相干时间——它们是信道的“DNA”
  3. 统计衰落模型：无LOS用瑞利衰落，有LOS用莱斯衰落，莱斯因子 $K$ 刻画LOS强度
  4. 标准信道模型：4G时代的EPA/EVA/ETU，5G NR的CDL/TDL，不同仿真目的选不同模型
  5. 多普勒效应：移动速度越快，相干时间越短，导频更新越频繁
  6. 信道→系统设计映射：时延扩展决定CP长度，多普勒扩展影响子载波间隔选择

信道建模是整个接收机算法设计的基石。下一讲，我们把这个抽象的信道模型“接”进真实系统：讲符号定时同步——接收机如何在一片噪声中找到OFDM符号的起点。

思考题

  1. 若某室内信道的RMS时延扩展为 $T_d=30\,\mathrm{ns}$，你设计的 OFDM 系统子载波间隔 $\Delta f$ 应不大于多少？（提示：参考相干带宽与子载波间隔的关系）
  2. 5G NR FR1在60 kHz子载波间隔下，单个OFDM符号时长约为 $T_s=1/\Delta f+T_{CP}$。若使用高铁场景（360 km/h，3.5 GHz），验证在一个符号持续时间内信道是否可以近似为时不变信道？
  3. 你的仿真代码使用ETU信道，移动速度设为5 km/h，但误把多普勒设置成了300 km/h的参数。从信道的时变性角度，这会带来什么问题？
