C/A 码扩频基带信号：数据 bit 采样、频谱与功率谱

这份笔记默认使用连续时间傅里叶变换的 Hz 频率约定：\(X(f)=\int x(t)e^{-j2\pi ft}dt\)。如果只看理想周期重复的 C/A 码，频域是间隔 1 kHz 的离散谱线；如果再乘上随机导航数据或截取有限长度观测，频谱会从理想线谱变成更接近工程仪器上看到的谱线展宽和起伏。

1. 数据 bit 的采样索引

设导航数据 bit 序列为：

\[ d[0],\ d[1],\ d[2],\ldots,\qquad d[m]\in\{+1,-1\} \tag{1} \]

数据位周期为 \(T_d\)。若从 \(t=0\) 开始计时，并且采用左闭右开的时间区间，则：

\[ D(t)=d[0],\quad 0\le t

\[ D(t)=d[1],\quad T_d\le t<2T_d \]

\[ D(t)=d[2],\quad 2T_d\le t<3T_d \tag{2} \]

因此，采样时刻 \(\tau_n\) 对应的数据位索引为：

\[ \boxed{ m_n=\left\lfloor\frac{\tau_n}{T_d}\right\rfloor } \tag{3} \]

采样时刻的数据值就是：

\[ \boxed{ D(\tau_n)=d[m_n] =d\left[\left\lfloor\frac{\tau_n}{T_d}\right\rfloor\right] } \tag{4} \]

如果接收机的本地时间不是从数据 bit 边界开始，而是存在数据起始时刻 \(t_{d,0}\)，则应写成：

\[ m_n=\left\lfloor\frac{\tau_n-t_{d,0}}{T_d}\right\rfloor \tag{5} \]

实际 ADC 采样常写成 \(\tau_n=t_{\text{start}}+nT_s\)，于是：

\[ m_n= \left\lfloor \frac{t_{\text{start}}+nT_s-t_{d,0}}{T_d} \right\rfloor \tag{6} \]

码片速率 \(R_c=1.023\,\text{MHz}\)，所以 \(T_c=1/R_c\)。
码长 C/A 码一个周期 \(N=1023\) 个码片。
码周期 \(T_{\text{code}}=NT_c=1\,\text{ms}\)。
数据位 GPS L1 C/A 导航数据通常为 50 bps，即 \(T_d=20\,\text{ms}=20T_{\text{code}}\)。
边界约定很重要

用左闭右开区间时，刚好落在 \(\tau_n=kT_d\) 的样点属于第 \(k\) 个 bit，而不是第 \(k-1\) 个 bit。仿真中若用浮点数计算时间，最好对边界误差做统一处理，避免同一个 bit 边界在不同模块里被分到不同索引。

2. C/A 码扩频基带模型

定义码片矩形脉冲：

\[ p(t)= \begin{cases} 1, & 0\le t

设 C/A 码片为 \(c_k\in\{+1,-1\}\)。如果先讨论无导航数据、只看周期重复的 C/A 码，则令：

\[ c_{k+N}=c_k,\qquad N=1023 \tag{8} \]

连续时间 C/A 码波形可以写成：

\[ C(t)=\sum_{k=-\infty}^{\infty}c_k p(t-kT_c) \tag{9} \]

把码片值放到加权冲激串上：

\[ a(t)=\sum_{k=-\infty}^{\infty}c_k\delta(t-kT_c) \tag{10} \]

利用冲激的平移筛选性质：

\[ \delta(t-kT_c)*p(t)=p(t-kT_c) \tag{11} \]

所以：

\[ \boxed{ C(t)=a(t)*p(t) } \tag{12} \]

乘上导航数据 bit 后，BPSK 扩频基带信号可以写成：

\[ s(t)=D(t)C(t) \tag{13} \]

如果接收机只在一个数据 bit 内做短时间相干积分，\(D(t)\) 在该窗口内近似是常数 \(+1\) 或 \(-1\)，因此它只改变整体符号，不改变局部 C/A 码频谱形状。若积分窗口跨过数据 bit 翻转点，符号变化会造成相干积分抵消。

3. 矩形码片给出的 sinc 包络

由卷积定理：

\[ C(f)=A(f)P(f) \tag{14} \]

其中矩形码片脉冲的傅里叶变换为：

\[ P(f)=\int_0^{T_c}e^{-j2\pi ft}\,dt \tag{15} \]

计算得：

\[ \begin{aligned} P(f) &=\left.\frac{e^{-j2\pi ft}}{-j2\pi f}\right|_0^{T_c}\\ &=T_c e^{-j\pi fT_c}\operatorname{sinc}(fT_c) \end{aligned} \tag{16} \]

这里采用归一化 sinc 定义：

\[ \operatorname{sinc}(x)=\frac{\sin(\pi x)}{\pi x} \tag{17} \]

所以码片脉冲带来的幅度包络是：

\[ |P(f)|=T_c|\operatorname{sinc}(fT_c)| \tag{18} \]

功率包络是：

\[ |P(f)|^2=T_c^2\operatorname{sinc}^2(fT_c) \tag{19} \]

sinc 零点由 \(\operatorname{sinc}(fT_c)=0\) 决定，即：

\[ fT_c=\pm1,\pm2,\pm3,\ldots \tag{20} \]

因此：

\[ \boxed{ f=\pm\frac{n}{T_c}=\pm nR_c } \tag{21} \]

对 GPS L1 C/A 码，\(R_c=1.023\,\text{MHz}\)，所以主包络零点位于：

\[ \pm1.023\,\text{MHz},\quad \pm2.046\,\text{MHz},\quad \pm3.069\,\text{MHz},\ldots \tag{22} \]

第一对零点之间的零到零带宽是 \(2.046\,\text{MHz}\)。这就是常说 C/A 码主瓣大约占据 \(\pm1.023\,\text{MHz}\) 的原因。

4. 周期 C/A 码的离散谱线

若 C/A 码严格周期重复，一个周期包含 \(N=1023\) 个码片：

\[ T_{\text{code}}=NT_c=1\,\text{ms} \tag{23} \]

对应基频为：

\[ \boxed{ f_0=\frac{1}{T_{\text{code}}}=1\,\text{kHz} } \tag{24} \]

因为 \(a(t)\) 是周期信号，所以它的频谱是离散谱线：

\[ A(f)=\sum_{m=-\infty}^{\infty}A_m\delta(f-mf_0) \tag{25} \]

系数 \(A_m\) 由一个周期内 C/A 码序列的 DFT 决定。更严格地写，在 Hz 频率约定下：

\[ \boxed{ A_m=\frac{1}{T_{\text{code}}} \sum_{k=0}^{N-1}c_k e^{-j2\pi mk/N} } \tag{26} \]

如果只关心谱线相对形状，也常写成比例关系：

\[ A_m\propto \sum_{k=0}^{N-1}c_k e^{-j2\pi mk/N} \tag{27} \]

代入 \(C(f)=A(f)P(f)\)，得到：

\[ \boxed{ C(f)= \sum_{m=-\infty}^{\infty} A_mP(mf_0)\delta(f-mf_0) } \tag{28} \]

因此理想周期 C/A 码的频谱有两个层次：

  • 谱线位置固定在 \(f=mf_0\)，间隔为 \(f_0=1\,\text{kHz}\)。
  • 谱线复幅度为 \(A_mP(mf_0)\)，既包含码序列 DFT，也包含码片矩形脉冲的 sinc 包络。
为什么零点刚好对齐

\(R_c=1.023\,\text{MHz}=1023\,\text{kHz}=1023f_0\)。因此 sinc 包络的第一零点对应第 \(m=\pm1023\) 条谱线，第二零点对应第 \(m=\pm2046\) 条谱线。矩形码片脉冲会把这些位置上的谱线压到零。

5. 功率谱线与 sinc² 包络

周期信号是功率信号。对周期信号来说，功率谱密度不是普通连续函数，而是一组带权重的冲激谱线。若：

\[ C(t)=\sum_{m=-\infty}^{\infty}C_m e^{j2\pi mf_0t} \tag{29} \]

则功率谱密度可以写成：

\[ S_C(f)=\sum_{m=-\infty}^{\infty}|C_m|^2\delta(f-mf_0) \tag{30} \]

由式 (28) 可知：

\[ C_m=A_mP(mf_0) \tag{31} \]

所以：

\[ \boxed{ S_C(f)= \sum_{m=-\infty}^{\infty} |A_m|^2|P(mf_0)|^2\delta(f-mf_0) } \tag{32} \]

再代入矩形脉冲的功率包络：

\[ \boxed{ |C_m|^2= |A_m|^2T_c^2\operatorname{sinc}^2(mf_0T_c) } \tag{33} \]

这条式子是理解 C/A 码功率谱的核心：\(\operatorname{sinc}^2\) 只给出外层趋势，\(|A_m|^2\) 决定每一根谱线在趋势线上下的具体起伏。

6. 为什么实际频谱不是光滑 sinc

如果只画 \(|P(f)|\) 或 \(|P(f)|^2\)，会得到一条光滑的 sinc 或 sinc² 曲线。但真正的周期 C/A 码频谱不是这条曲线本身，而是离散谱线：

\[ C(f)= \sum_m A_mP(mf_0)\delta(f-mf_0) \tag{34} \]

每根谱线在 \(f=mf_0\) 处的复幅度是：

\[ C_m=A_mP(mf_0) \tag{35} \]

幅度为：

\[ |C_m|=|A_m|T_c|\operatorname{sinc}(mf_0T_c)| \tag{36} \]

功率为：

\[ |C_m|^2 =|A_m|^2T_c^2\operatorname{sinc}^2(mf_0T_c) \tag{37} \]

因此需要把两个因素分开：

矩形码片脉冲 \(P(f)\) 决定外层 sinc 包络，零点在 \(\pm nR_c\)。
码序列 DFT \(A_m\) 决定不同 1 kHz 谱线之间的起伏，是具体 PRN 序列造成的。
实际线谱 \(C_m\) 是二者相乘：\(C_m=A_mP(mf_0)\)。

对于伪随机码，\(|A_m|^2\) 在很多频点上会表现出类似随机的细碎起伏，因此单根谱线不会严格落在一条光滑的 sinc² 曲线上。只有从平均趋势、包络或频率平滑后的结果看，才会明显呈现 sinc² 控制的主瓣和旁瓣结构。

观察对象 数学形式 物理含义
码片脉冲 \(P(f)=T_ce^{-j\pi fT_c}\operatorname{sinc}(fT_c)\) 一个矩形码片自身的频谱。
周期码序列 \(A_m\propto\sum_{k=0}^{N-1}c_ke^{-j2\pi mk/N}\) 一个周期内 C/A 码值的 DFT。
频谱线幅度 \(|C_m|=|A_m||P(mf_0)|\) DFT 起伏乘上 sinc 外层包络。
功率谱线 \(|C_m|^2=|A_m|^2T_c^2\operatorname{sinc}^2(mf_0T_c)\) 线谱功率围绕 sinc² 包络起伏。

7. 导航数据和有限观测的影响

上面的离散线谱结论有一个重要前提：C/A 码无限长、严格周期重复，而且没有随机导航数据改变符号。工程频谱通常还会受到以下因素影响。

7.1 导航数据 bit 会破坏严格 1 ms 周期性

完整基带信号是：

\[ s(t)=D(t)C(t) \tag{38} \]

频域中，时域相乘对应频域卷积：

\[ S(f)=D(f)*C(f) \tag{39} \]

如果 \(D(t)\) 在观测时间内恒定为 \(+1\) 或 \(-1\)，则它只改变整体符号；如果 \(D(t)\) 是 50 bps 导航数据，并且在长时间内随机翻转，那么它会把原来理想的周期码线谱进行调制和展宽。直观上，数据 bit 让每 20 ms 可能发生一次整体符号翻转，严格的 1 ms 周期重复不再对无限时间成立。

7.2 有限观测窗口会造成谱线展宽和泄漏

频谱仪或 FFT 只能观察有限长度信号。设观察窗口为 \(w_T(t)\)，实际拿去变换的是：

\[ x_T(t)=x(t)w_T(t) \tag{40} \]

因此频域为：

\[ X_T(f)=X(f)*W_T(f) \tag{41} \]

理想冲激谱线会被窗口频谱 \(W_T(f)\) 展宽成主瓣和旁瓣。如果 FFT 频率栅格没有刚好对齐 1 kHz 谱线，或者观察长度不是码周期的整数倍，还会出现更明显的频谱泄漏。

7.3 前端滤波、采样和非理想码片形状也会改变包络

真实接收链路还包含射频/中频滤波器、ADC 采样、量化、自动增益控制、载波 Doppler 和本地码相位误差。这些因素会继续改变观测到的谱线高度。特别是前端带宽较窄时，sinc 外层包络还会再乘上接收机滤波器响应。

一句话区分

理想数学模型说明“C/A 周期码的线谱在哪里、外层包络是什么”；工程观测结果则是这个线谱再经过导航数据调制、有限窗口卷积、前端滤波和离散采样后的结果。

8. 仿真采样时的实现细节

在离散仿真中，若采样率为 \(F_s\)，采样间隔 \(T_s=1/F_s\)，第 \(n\) 个样点时间为：

\[ \tau_n=t_{\text{start}}+nT_s \tag{42} \]

则数据 bit 索引可写成：

\[ m_n=\left\lfloor\frac{\tau_n-t_{d,0}}{T_d}\right\rfloor \tag{43} \]

码片索引可写成：

\[ k_n= \left\lfloor \frac{(\tau_n-t_{c,0})\bmod T_{\text{code}}}{T_c} \right\rfloor \tag{44} \]

于是基带扩频序列样点为：

\[ s[n]=d[m_n]c[k_n] \tag{45} \]

若还要加入载波 Doppler、初始相位和噪声，可以在此基础上继续乘上复载波并加噪：

\[ r[n]=A\,d[m_n]c[k_n]e^{j(2\pi f_D\tau_n+\phi_0)}+w[n] \tag{46} \]

实现点 建议
bit 边界 明确使用左闭右开区间，并统一处理浮点边界误差。
码片索引 先对 \(T_{\text{code}}\) 取模，再除以 \(T_c\) 取 floor，最后限制在 \(0,\ldots,1022\)。
符号表示 频谱推导中用 \(\{+1,-1\}\) 最方便；若原始码是 \(\{0,1\}\)，需要先映射为双极性码。
相干积分 积分窗口跨越未知数据 bit 翻转时会损失相干增益，捕获和跟踪中常需要数据擦除或限制积分长度。
FFT 观察频谱 选择码周期整数倍的观测长度可以减少泄漏；加窗会降低旁瓣但也会加宽主瓣。

9. 一页总结

数据 bit 采样索引：

\[ m_n=\left\lfloor\frac{\tau_n}{T_d}\right\rfloor,\qquad D(\tau_n)=d[m_n] \]

码片矩形脉冲：

\[ P(f)=T_ce^{-j\pi fT_c}\operatorname{sinc}(fT_c) \]

周期 C/A 码频谱：

\[ C(f)= \sum_m A_mP(mf_0)\delta(f-mf_0), \qquad f_0=1\,\text{kHz} \]

谱线系数：

\[ A_m=\frac{1}{T_{\text{code}}} \sum_{k=0}^{N-1}c_k e^{-j2\pi mk/N} \]

功率谱线：

\[ |C_m|^2= |A_m|^2T_c^2\operatorname{sinc}^2(mf_0T_c) \]

最需要记住的是：C/A 码的频谱不是一条连续光滑的 sinc 曲线，而是一组 1 kHz 间隔的离散谱线；sinc 或 sinc² 只是矩形码片给出的外层包络，具体每根谱线还要乘上 C/A 码一个周期内的 DFT 系数。导航数据、有限观测窗口和接收机前端会进一步改变工程上看到的频谱形状。
