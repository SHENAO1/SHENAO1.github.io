为什么方波只含奇次谐波就只在奇数倍的基频上出现频率分量？这相对于是半波反对称函数，如果是半波对称函数呢？会怎么样？

## 🤖 Gemini

这两个问题问得很深刻，直接切中了信号频域分析的核心——**时域波形的对称性决定了频域谱线的分布规律**。

我们可以从傅里叶级数（Fourier Series）的数学本质来拆解这两个问题。


## 一、 为什么“只含奇次谐波”就只在奇数倍基频上出现频率分量？
对于一个周期为 $T_s$、基频为 $f_s = 1/T_s$ 的周期信号 $x(t)$，它的双边傅里叶级数展开式为：


$$x(t) = \sum_{n=-\infty}^{+\infty} X_n e^{j 2\pi n f_s t}$$
其中 $X_n$ 是第 $n$ 次谐波的复振幅，对应的频率分量就在 $f = n f_s$ 处。


### 1. 奇次谐波的数学体现
所谓的“方波只含奇次谐波”，意味着当 $n$ 为偶数（$n = \pm 2, \pm 4, \dots$）时，傅里叶系数 $X_n = 0$。只有当 $n$ 为奇数时，$X_n \neq 0$。


### 2. 谱线位置与间距
既然偶数项全为 0，那么频域中剩下的谱线自然就只能排在奇数倍的基频上了：


- **谱线位置**：$f = \pm 1f_s, \pm 3f_s, \pm 5f_s, \dots$，统一写成表达式就是 $f = \pm(2k+1)f_s \quad (k=0,1,2,\dots)$。
- **谱线间隔**：相邻的两个奇次谐波（比如 $3f_s$ 和 $1f_s$）之间的距离，自然就是 $3f_s - 1f_s = 2f_s$。

这就是为什么在 BOC 调制中，副载波（方波）的能量会从中心频点搬移，跳过直流（$0$）和偶数倍频点，直接分裂并聚集在 $\pm f_s$ 附近的原因。


## 二、 如果是“半波对称（半波顺对称）”函数会怎么样？
你提到的方波属于**半波反对称（奇谐对称）**，满足 $x(t + T_s/2) = -x(t)$。
如果我们把条件反过来，变成**半波对称（偶谐对称）**，即满足：


$$x(t + \frac{T_s}{2}) = x(t)$$
我们来看看它的傅里叶系数 $X_n$ 会发生什么变化。


### 1. 数学推导
傅里叶系数的计算公式为：


$$X_n = \frac{1}{T_s} \int_{0}^{T_s} x(t) e^{-j 2\pi n f_s t} dt$$
把积分区间拆成前后半周期：


$$X_n = \frac{1}{T_s} \left[ \int_{0}^{T_s/2} x(t) e^{-j 2\pi n f_s t} dt + \int_{T_s/2}^{T_s} x(t) e^{-j 2\pi n f_s t} dt \right]$$
对后半部分积分进行换元，令 $\tau = t - T_s/2$（即 $t = \tau + T_s/2$），并代入**半波对称条件** $x(\tau + T_s/2) = x(\tau)$：


$$\int_{0}^{T_s/2} x(\tau + \frac{T_s}{2}) e^{-j 2\pi n f_s (\tau + \frac{T_s}{2})} d\tau = \int_{0}^{T_s/2} x(\tau) e^{-j 2\pi n f_s \tau} \cdot e^{-j n \pi} d\tau$$
因为 $e^{-j n \pi} = (-1)^n$，把两部分积分合并回来：


$$X_n = \frac{1}{T_s} \left[ 1 + (-1)^n \right] \int_{0}^{T_s/2} x(t) e^{-j 2\pi n f_s t} dt$$

### 2. 得出结论：只含偶次谐波
由上面的表达式 $\left[ 1 + (-1)^n \right]$ 可以直接看出：


- 当 $n$ 为**奇数**时：$1 + (-1) = 0 \implies X_n = 0$ （**奇次谐波全部消失**）
- 当 $n$ 为**偶数**时：$1 + 1 = 2 \implies X_n \neq 0$ （**只保留偶次谐波和直流分量**）

因此，如果是半波对称函数，它的频谱是一串只存在于 **$f = \pm 2k f_s \quad (k=0,1,2,\dots)$** 上的谱线，线谱间隔同样为 $2f_s$，但频点变成了 $0, \pm 2f_s, \pm 4f_s, \dots$。


## 三、 直观的物理本质理解
其实不用复杂的数学公式，从物理和直观上也非常好理解：


> 如果一个信号满足半波对称 $x(t + T_s/2) = x(t)$，这在本质上意味着**它根本不需要等到 $T_s$ 时间才重复，它在 $T_s/2$ 的时间内就已经完成了一个完整的周期循环**。
>
>
既然它的真正周期是 $T_{\text{true}} = T_s/2$，那么它的真正基频其实是：


$$f_{\text{true}} = \frac{1}{T_{\text{true}}} = \frac{2}{T_s} = 2f_s$$
既然真正的基频已经是 $2f_s$ 了，它的光谱线自然只能出现在 $2f_s$ 的整数倍上（即 $0, 2f_s, 4f_s, 6f_s \dots$）。如果依然强行用原来的 $f_s$ 去度量它，它表现出来的就全都是“偶次谐波”了。

## 🤖 Gemini

这个问题问到了点子上。要解释为什么公式中除以的是码片周期 $T_c$（Chip duration）而不是副载波周期 $T_s$（Subcarrier period），我们需要从**信号的随机性来源**以及**功率谱密度的数学定义**两个层面来理解。

核心结论是：**因为该信号的“随机独立信息”是以 $T_c$ 为单位变化的，每一个独立的完整波形结构持续的时间是 $T_c$，而不是 $T_s$。**


## 1. 信号的数学模型与随机性来源
在卫星导航（如 GPS、Galileo、BeiDou）中，基带信号通常被建模为一个**平稳随机过程**。以 BOC 调制为例，信号可以表示为：


$$s(t) = \sum_{k=-\infty}^{+\infty} c_k \cdot p(t - k T_c)$$
其中：


- $c_k \in \{+1, -1\}$ 是伪随机噪声码（PRN Code）的码片，它们是**随机独立**的。
- $p(t)$ 是**单个码片内的脉冲波形**。它的持续时间正是 **$T_c$**。

对于 BOC 调制而言，这个 $p(t)$ 不再是普通的矩形脉冲，而是**在 $T_c$ 时间内充满了方波副载波**的复合波形。也就是说，副载波的变化已经被“包裹”在了 $p(t)$ 的内部。


## 2. 功率谱密度的推导本质（维纳-辛钦定理）
根据随机信号分析，对于这种由彼此独立的随机序列调制的信号，其归一化功率谱密度 $G(f)$ 的标准计算公式（由单个脉冲的能量谱推导平均功率谱）为：


$$G(f) = \frac{1}{T_{\text{symbol}}} |P(f)|^2$$
这里的 $T_{\text{symbol}}$ 指的是**符号周期（即随机变量保持不变的独立时间间隔）**。


- 在导航信号中，伪码每隔 $T_c$ 秒才会变成下一个随机的 $+1$ 或 $-1$。
- 因此，这里的独立符号周期 **$T_{\text{symbol}} = T_c$**。
- $P(f)$ 是对这整个持续时间为 $T_c$ 的单个复合脉冲 $p(t)$ 求傅里叶变换得到的结果：


$$P(f) = \int_{0}^{T_c} p(t) e^{-j 2\pi f t} dt$$

## 3. 为什么不能除以 $T_s$？
我们可以从以下两个视角来直观理解为什么除以 $T_s$ 是错误的：


### 视角一：内部的副载波是“确定性”的
在 $T_c$ 这段时间内，虽然副载波按 $T_s$ 的周期在正负翻转，但这个翻转规律是**完全确定**的（例如固定的每隔 $T_s/2$ 翻转一次）。它不带有任何新的随机信息。
既然在 $T_s$ 的边界上没有新的随机信号注入，那么 $T_s$ 就不能作为统计平均的时间基准。


### 视角二：能量与功率的量纲转换

- $P(f)$ 的物理意义是**单码片信号的频谱**，那么 $|P(f)|^2$ 就是**单码片信号的能量谱密度（单位：焦耳/赫兹）**。
- 功率谱密度的本质是：$\text{功率} = \frac{\text{能量}}{\text{时间}}$。
- 既然 $|P(f)|^2$ 承载的是一个长度为 $T_c$ 的波形的能量，要把它转化为平均功率，就必须除以这个波形实际对应的时间跨度 $T_c$。


## 总结
表格对比可以让你更清晰地看出它们的分工：

| 参数 | 物理意义 | 在公式 $G(f) = \frac{1}{T_c} |P(f)|^2$ 中的角色 |
| :--- | :--- | :--- |
| **$T_c$** | 码片周期（控制随机信息更新的节奏） | **作为分母**。用来将单个完整脉冲的“能量”转化为全时段的“平均功率”。 |
| **$T_s$** | 副载波周期（控制脉冲内部的细节纹理） | **隐藏在 $P(f)$ 内部**。它决定了 $P(f)$ 的具体形状（比如让谱线在何处分裂、产生奇次谐波）。 |
